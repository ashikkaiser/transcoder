POST /transcode Submit videos for transcoding

Request Body

{
  "videos": [
    {
      "id": 12345,
      "s3_key": "products/12345/video.mp4",
      "video_url": "https://cdn.example.com/video.mp4",
      "callback_url": "/api/transcode/callback"  // optional
    }
  ]
}

Response 200

{
  "accepted": [
    {
      "video_id": 12345,
      "job_id": "a1b2c3d4-...",
      "status": "queued",
      "created": true
    }
  ]
}

• callback_url is optional. Omit to use the base URL from Settings.

• If a job already exists for the same video_id, the existing job is returned (created: false).

• Max 100 videos per batch request.

POST /transcode/cancel Cancel a transcoding job

Request Body

{
  "video_id": 12345,
  "reason": "No longer needed"  // optional
}

Response 200

{
  "video_id": 12345,
  "status": "canceled",
  "message": "Job a1b2c3d4-... canceled"
}
